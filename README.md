# City Pulse

City Pulse is a modern digital platform designed to bridge the gap between citizens and city administration. It serves as a unified portal for reporting issues, accessing city information, paying bills, and earning rewards for civic engagement.

## Features

-   **Bilingual Support**: Full support for English and Kannada.
-   **City Information**: Real-time data on AQI, weather, traffic, and public facilities.
-   **Issue Reporting**: Easy-to-use interface for reporting garbage, potholes, and other civic issues.
-   **Bill Payments**: Secure payment gateway for electricity, water, and other utility bills.
-   **Rewards System**: Gamified experience where citizens earn points and badges for their contributions.
-   **Analytics Dashboard**: Visual insights into city improvements and issue resolution metrics.

## Getting Started

This project is built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

### Prerequisites

-   Node.js 18.17 or later.

### Installation

1.  Clone the repository or navigate to the project directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

-   **Framework**: Next.js 14
-   **Styling**: Tailwind CSS
-   **Icons**: Lucide React
-   **Language**: TypeScript
